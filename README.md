My IDE and terminal settings.

## Install

```shell
git clone https://github.com/<your_user>/<your_repository> ~/.config/nvim

# On Mac, install dependencies like Node. Figure out the equivalents for linux
./install_macos.sh

# Install kitty
# Then move the config
cp kitty.conf ~/.config/kitty/kitty.conf

# If remote & using tmux, copy tmux config
cp .tmux.conf ~/.tmux.conf

# Install copilot or whatever AI cli I'm using these days
```
